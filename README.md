- プロジェクト名

Obsidianのバスケ戦術メモをもとに、Claude Codeを使って講座構成と本文を整え、最終的にUTAGEへ格納するための制作プロジェクト。

- このプロジェクトの目的

・Obsidian内の知識を講座形式へ変換する
・Claude Codeで構成整理、本文作成、整形を進める
・自分のレビューを通して内容精度を上げる
・完成原稿をUTAGEへMCP経由で格納する

- 制作対象

・本プロジェクトでは、はじめに以降の章を制作対象とする
・はじめにの章はすでに作成完了済みであり、原則として再生成しない
・レッスン形式は文字、画像、動画を想定するが、最初の制作対象は文字コンテンツのみとする
・画像と動画の挿入は最終工程で人間が行う

- 想定読者

・高校生を教えているコーチ
・大学生を教えているコーチ
・プロ選手を教えているコーチ

- 作業の全体フロー

1. source に元メモを入れる
2. outline で講義構成を作る
3. draft で各章の本文を作る
4. review で人間が修正指示を入れる
5. publish に確定版を置く
6. Claude CodeがUTAGEへ格納する

- フォルダの意味

source
Obsidianから抜き出した元資料

outline
講義の章立て、節構成、学習順序

draft
Claude Codeが作成した下書き

review
人間の修正メモ、差分指示

publish
UTAGE投入用の確定原稿

- 優先方針

・最初は文字だけで講座を成立させる
・画像と動画を前提にしすぎない
・まず1章ずつ作る
・UTAGE投入は確定版のみ
・内容の正確性と独自性は人間が最終責任を持つ
