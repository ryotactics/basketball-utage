# MODULE 1 - Lesson 2｜アドバンテージの作り方


## 前のレッスンからのつながり

前のレッスンで、オフェンスの根本は「アドバンテージを作って増大させること」だと学びました。

では具体的に、そのアドバンテージはどうやって作るのか。ここが本レッスンのテーマです。


## アドバンテージ生成の2つの軸

アドバンテージを作る方法は、大きく2つの軸で整理できます。

### 1. 人を動かす
### 2. ボールを動かす

この2つが、オフェンスがディフェンスを崩すための根本的な手段です。

### 人を動かす

「人を動かす」とは、ボールを持っていない選手・ボールを持っている選手の位置や動きを使ってディフェンスの構造を崩すアプローチです。

- スクリーンで壁を作ってディフェンスを引き剥がす
- カットでディフェンスの視野の外から飛び出す
- スペーシングを変化させてディフェンスの立ち位置を揺さぶる

ディフェンスは必ず誰かをマークしているので、人が動けばディフェンスも動かざるを得ません。その動きの中にズレを生み出すのが「人を動かす」という手段です。

### ボールを動かす

「ボールを動かす」とは、パスやドリブルでボールそのものの位置を変化させることでディフェンスを崩すアプローチです。

- 素早いボールムーブで認知を遅らせる
- ドライブでヘルプを引き出し、スペースを作る
- スキップパスで逆サイドを一気に使う

ディフェンスはボールに対して反応するので、ボールを動かせばディフェンスも反応して動きます。その反応の速さを超えてボールを回すことで、ディフェンスのズレを突きます。

### 両方を組み合わせる

実戦では、「人を動かす」と「ボールを動かす」は常に併用されます。

例えばPNRは、スクリーナーが動き（人）、ハンドラーがドリブルする（ボール）という2軸の複合です。オフボールスクリーンの間もボールはスイングされ、パス＆カッティングはボールと人が交互に動きます。

コーチとして意識すべきは、「自分たちのオフェンスは、人を動かすことで作っているのか、ボールを動かすことで作っているのか、それとも両方か」を言語化できることです。


## 具体的な3つのアクション系統

「人を動かす / ボールを動かす」という軸に加えて、実際にアドバンテージを作るためのアクションは、大きく3つの系統に分類できます。

### 1. オフボールスクリーンアクション（Off Ball Screen Action）

ボールを持っていない選手にスクリーンをかけて、その選手の動きを使ってアドバンテージを作る方法です。

代表例：Pindown / Flare / Stagger / Elevator / AI

ディフェンスはボール以外のエリアでの対応も迫られ、視野が分散します。そこで生まれる一瞬のズレがアドバンテージになります。

### 2. オンボールスクリーンアクション（On Ball Screen Action）

ボールを持っている選手にスクリーンをかける方法です。代表的なのはPNR（ピック&ロール）です。

スクリーナーとハンドラーの2人が同時に関わることで、ディフェンス側は「どちらを守るか」「どう連携するか」の複雑な判断を迫られます。

PNRはアドバンテージを作る最もシンプルで強力な手段であり、現代バスケの中核をなすアクションです。

### 3. パス＆カッティング／ドライブ＆キック（Pass & Cutting / Drive & Kick）

スクリーンに頼らず、パスやドライブでディフェンスを崩す方法です。

- パス＆カッティング — パスを出したあと、ボールなしでカットすることでディフェンスのズレを突く
- ドライブ＆キック — ドライブでヘルプを引き出し、空いたスペースにキックアウトする

スクリーンが使えない状況でも、パスやドライブの質が高ければアドバンテージは作れます。特にドライブ＆キックはシンプルですが、ヘルプの引き出し方と次のパスの選択次第で連鎖を起こしやすい手段です。


## 3系統のどれを軸にするかを決める

チームとしてアドバンテージの作り方を整理するには、「自チームはどの系統を軸にするのか」を決める必要があります。

例：
- ハンドラーが強い → オンボールスクリーン（PNR）を軸に据える
- シューターが複数いる → オフボールスクリーンを軸に据える
- アスレチックな選手が多い → パス＆カッティング／ドライブ＆キックを軸に据える

これは前のMODULE 0で触れた「戦略」と「コンセプト」の話と繋がります。自チームの強みに合った作り方を選ぶことで、アドバンテージが再現性高く生まれるようになります。


## 「作れるアドバンテージ」と「使えるアドバンテージ」は違う

アドバンテージを作る方法を整理するとき、もう一段踏み込んで見たいのは、そのアドバンテージを自チームが使えるかです。

たとえば、PNRでDropに対してPull Up Mid Rangeを作れるとしても、そのショットを安定して打てるHandlerがいなければ、実戦では相手に許容されます。Off Ball Screenでシューターを外せても、パス角度やキャッチの姿勢が悪ければ、Closeoutが間に合います。DriveでHelpを引き出せても、Kick先の選手が次の判断をできなければ、アドバンテージはそこで消えます。

つまり、戦術選択では「形として何を作れるか」だけでなく、「その後に誰が何をできるか」まで見る必要があります。

指導現場では、次の問いが役立ちます。

### 1. このアクションで最初に誰が優位を得るのか
### 2. その選手は、その優位を得点・パス・次のアクションへ変換できるか
### 3. 変換できない場合、周囲の選手が出口になれるか
### 4. 相手が守り方を変えたとき、同じ入口から別の優位を作れるか

この問いを持つと、戦術が「有名だから使うもの」ではなく、「自チームの能力と接続して選ぶもの」になります。
