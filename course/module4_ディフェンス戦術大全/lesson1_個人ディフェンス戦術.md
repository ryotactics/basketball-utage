# MODULE 4 - Lesson 1｜個人ディフェンス戦術


## 前のモジュールからのつながり

## MODULE 3では、ディフェンス原論として、

- ディフェンスの目的（何を守るのか）
- ディフェンスの種類
- ディフェンスコンセプト設計
- ヘルプシステムの原則

を扱いました。

MODULE 4からは、現場で使う具体的なディフェンス戦術に入ります。まずは全ての土台となる個人ディフェンス戦術からです。

チームディフェンスがどれほど優れていても、個人のディフェンス精度が低ければ構造そのものが崩れます。逆に、個人ディフェンスが整っているチームは、シンプルなチームディフェンスでも機能します。


## このレッスンの目的

- 個人ディフェンスの原則を整理する
- オンボール／オフボールそれぞれの役割を理解する
- 個の技術がチームディフェンス全体の前提になることを言語化する


## 原理（MODULE 1／MODULE 3との接続）

個人ディフェンスは、オフェンス側のプレーの4原則（視野・認知・予測・重心） に逆から対応します。

- オフェンスが視野を突いてくる → ディフェンスは視野と位置取りで守る
- オフェンスが認知を揺さぶる → ディフェンスは判断スピードで守る
- オフェンスが予測を外す → ディフェンスは反応の幅で守る
- オフェンスが重心を崩す → ディフェンスはスタンスとバランスで守る

つまり個人ディフェンスは、オフェンス原理の裏返しとして体系化できます。


## 本文 — ユーザー執筆

このセクションでは、以下の観点から解説します。

### 1. オンボールディフェンス
- スタンス、ディスタンス、角度、手の位置
- On Ball Direction：どちらの手・どちらのスペースへ誘導するか
- ボールプレッシャーの強度設計：奪いにいくのか、時間を削るのか、侵入を止めるのか
- ドライブ対応：Contain、Force Direction、Chest、Slide、Recovery
- コンテストの質：飛ぶ／飛ばない、Fly-by、Fly-by Cancel、Closeout後の2歩目

### 2. オフボールディフェンス
- ボールサイド／ヘルプサイドのポジショニング
- Gap（ギャップ）の取り方
- Denial（デナイ）／Off Ball Close Outの判断
- スクリーンへのリアクション（ファイト／スライド／スイッチ）
- Ball Line、One Pass Away、Help Side、Low Manの位置関係
- Top Lock、Deny、Chaseなど、相手のオフボールアクションへの個人対応
- Stunt、Dig、Tagの強度と戻り方
- 自分のマークとボールの両方を見る視野の作り方

### 3. ディフェンスの個人スキル要素
- フットワーク
- 手の使い方
- ボックスアウト
- リバウンドポジショニング
- CloseoutからContainへ移る技術
- Contactを受けた後のバランス回復
- ファウルを避けながらプレッシャーをかける技術

### 4. チームディフェンスとの接続
- 個人のOn Ball Directionが、後ろのHelp位置を決める
- 個人が抜かれたとき、どのHelp / Rotationが発動するかをセットで扱う
- CloseoutやBox Outの精度が、チームのローテーション完了条件になる
- 個人技術を「1on1の上手さ」ではなく、チーム構造を成立させる部品として書く


## コーチが意識すべきこと

- 自チームは個人ディフェンスの強度基準を言語化できているか
- 選手は「どこに立つか」「どこまで寄るか」の判断基準を共通認識として持っているか
- 個人ディフェンスの反復練習が、チームディフェンスの練習と切り離されていないか

個人ディフェンスは技術であり、習慣であり、体力でもあります。戦術より前に、まずこの層の精度が勝負を決めます。
