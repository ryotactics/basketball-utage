# MODULE 4 - Lesson 2｜チームディフェンス戦術


## 前のレッスンからのつながり

前のレッスンでは、チームディフェンスの土台となる個人ディフェンスを扱いました。

このレッスンでは、個人ディフェンスを前提として、5人がどう連動するか——チームディフェンス戦術に入ります。

MODULE 3で扱った「ディフェンスコンセプト設計」と「ヘルプシステムの原則」を、実際の戦術レベルで具体化する内容です。


## このレッスンの目的

- チームディフェンスの主要な選択肢を整理する
- マンツーマン／ゾーン／ハイブリッドそれぞれの原理を理解する
- 自チームのディフェンスコンセプトを選ぶための判断軸を持つ


## 原理（MODULE 3との接続）

チームディフェンスは、5人の連動で以下を防ぐ構造です。

- リム近辺の高確率シュート
- オープン3P
- イージーなアウトナンバー

そのために、「誰がどこをカバーするか」「どうローテーションするか」のシステムを選びます。


## 本文 — ユーザー執筆

このセクションでは、以下の観点から解説します。

### 0. チームディフェンスを構成する部品
- On Ball Defense：方向づけ、圧力、侵入制限
- Help Defense：Gap、Ball Line、Tag、Low Man、Nail Help
- Rotation：X-Out、Peel、Scramble、Weak Side Rotation
- Closeout：飛ばすのか、止めるのか、打たせるのか
- Defensive Rebound：最後にポゼッションを終わらせる設計

### 1. マンツーマンディフェンス
- Pack Line / Aggressive Man / Switch All
- ヘルプローテーションの原則
- スクランブル（崩れた後の立て直し）
- Ball Pressure型、Gap型、Deny型、Switch型で何を優先して守るか
- Helpをどこから出し、どこを捨てるか
- Mismatchが起きたとき、Switch Backするのか、そのまま守るのか

### 2. ゾーンディフェンス
- 2-3 / 3-2 / 1-3-1 / 1-2-2
- 各ゾーンの強み・弱み
- ゾーンにおけるヘルプ概念とマッチアップゾーン
- 守るエリア、誘導するショット、リバウンド責任をセットで書く
- ゾーンを「守れないから使う逃げ」ではなく、相手の認知とテンポを崩す手段として扱う

### 3. ハイブリッド／ジャンクディフェンス
- Box and 1 / Triangle and 2
- マッチアップゾーン
- シチュエーション限定のジャンク運用
- エリートスコアラー対策、時間帯限定、相手の準備不足を突く使い方
- どこまでを約束事にし、どこから通常ディフェンスへ戻るか

### 4. チームディフェンス選定の判断軸
- 自チームの選手特性（サイズ、機動力、IQ）
- 相手チームの強み／弱み
- 試合のContext（点差、時間、ファウル）
- 何を消し、何を許容するか
- 基本ディフェンス、サブディフェンス、緊急用ディフェンスを分けて持つ
- 変更するときは、個人対応 → マッチアップ変更 → カバレッジ変更 → Help / Rotation変更 → システム変更の順で考える


## ディフェンス選択はトレードオフである

どのディフェンスを選んでも、何かを得る代わりに何かを諦める構造になっています。

| ディフェンス | 強み | 弱み（諦めること） |
| --- | --- | --- |
| マンツーマン | 個の責任が明確、相手の動きに密着できる | 個の力負けが直接失点につながる、スクリーンで崩されやすい |
| ゾーン | スクリーンへの対応が楽、体力温存 | 外側のシュートを与えやすい、ギャップを突かれる |
| Switch All | PNRへの対応がシンプル | ミスマッチを生んで逆用される |
| プレス | 相手のリズムを崩せる、速いアタックへの対応 | 体力消耗、プレスブレイクされると数的不利になる |

「完璧なディフェンス」は存在しません。コーチが問うべきは「このトレードオフを今の対戦相手・自チームの特性に照らして受け入れられるか」です。


## コーチが意識すべきこと

- 自チームの基本ディフェンスは何か、それはなぜその選択なのか
- 相手や状況に応じたサブディフェンスを何種類持っているか
- 各ディフェンスの切り替えの合図が決まっているか

チームディフェンスは「一つに決めて貫く」ではなく、「基本 + 切り札」の構造で持つと、試合中の対応力が上がります。
