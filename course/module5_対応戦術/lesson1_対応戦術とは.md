# MODULE 5 - Lesson 1｜対応戦術とは

---

## 前のモジュールからのつながり

MODULE 2ではオフェンス戦術の各論を、MODULE 3・4ではディフェンス原論と各論を扱いました。

ここまでは、**自チームがどう攻めるか／どう守るか**を扱ってきました。

MODULE 5では視点を変えます。テーマは**「相手がしてくることに対して、自分たちはどう対応するか」**——対応戦術です。

---

## 対応戦術とは

試合は、一方的に自分たちの戦術を出すだけでは決まりません。

- **相手がこのディフェンスをしてきたら、どのオフェンスで崩すか**
- **相手がこのオフェンスをしてきたら、どのディフェンスで止めるか**
- **相手が途中でアジャストしてきたら、どう対抗するか**

この**相互作用**の中での判断が、実戦での「対応戦術」です。

MODULE 1〜4で学んだ知識は、この対応戦術を**根拠を持って選べる**ようにするための基盤です。

## コーチが意識すべきこと

- 「自分たちの戦術を出す」と「相手の戦術に対応する」は**別のスキル**
- 対応戦術は、**相手を観察・分析する力**と**手持ちの引き出しの広さ**で決まる
- MODULE 1で学んだ**優位設計の切り替え**が、対応戦術の根幹になる
