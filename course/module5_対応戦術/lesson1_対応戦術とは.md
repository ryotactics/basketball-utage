# MODULE 5 - Lesson 1｜対応戦術とは


## 前のモジュールからのつながり

MODULE 2ではオフェンス戦術の各論を、MODULE 3・4ではディフェンス原論と各論を扱いました。

ここまでは、自チームがどう攻めるか／どう守るかを扱ってきました。

MODULE 5では視点を変えます。テーマは「相手がしてくることに対して、自分たちはどう対応するか」——対応戦術です。


## 対応戦術とは

試合は、一方的に自分たちの戦術を出すだけでは決まりません。

- 相手がこのディフェンスをしてきたら、どのオフェンスで崩すか
- 相手がこのオフェンスをしてきたら、どのディフェンスで止めるか
- 相手が途中でアジャストしてきたら、どう対抗するか

この相互作用の中での判断が、実戦での「対応戦術」です。

MODULE 1〜4で学んだ知識は、この対応戦術を根拠を持って選べるようにするための基盤です。


対応は「別の戦術を足すこと」ではない

対応戦術というと、新しいセットや特別なカウンターを追加することだと思われがちです。

しかし、実戦で最初に考えるべきなのは、今ある手札の中で何を変えれば対応できるかです。

変更には段階があります。


### 1. 頻度を変える
### 2. 配置を変える
### 3. 起点となる選手を変える
4. Option / Readの優先順位を変える
### 5. Counterを出す
### 6. 別コンセプトへ移る

たとえば、相手がPNRをDropで守っているからといって、すぐに全く別のオフェンスへ移る必要はありません。まずはPull-up、Short Roll、Pop、Weak SideのLiftなど、既存のPNR内で何を優先するかを変える。次に、SpainやVeerのようにActionでCoverageを揺さぶる。それでも重いなら、PostやOff Ball Screenなど別コンセプトへ移る。

この順番を持っていると、対応が場当たり的になりません。

対応戦術の質は、引き出しの数だけで決まりません。どの順番で、どの大きさの変更を行うかで決まります。小さい変更で解決できるなら、それが最も選手の負担が少なく、再現性の高い対応です。


対応を見る3つの問い

試合中に対応を考えるときは、次の3つを順番に見ます。

1つ目は、相手は何を消しているかです。Dropならリムを消している。Top LockならCatchを消している。Switchならセパレーションを消している。この「消しているもの」を見ないと、相手の守り方の目的がわかりません。

2つ目は、その代わりに何を与えているかです。すべてを消せる守り方はありません。DropはPull-upやShort Rollを与え、Top LockはBack Cutを与え、SwitchはミスマッチやSlipを与えます。

3つ目は、自チームのロスターで、その与えられたものを使えるかです。Pull-upを与えられていても、打てるHandlerがいなければ軸にできません。Short Rollを与えられていても、判断できるBigがいなければ機能しません。

対応戦術は、「相手の弱点」だけで決めるものではありません。相手が与えているものと、自チームが実行できるものが重なった場所を選ぶ必要があります。

## コーチが意識すべきこと

- 「自分たちの戦術を出す」と「相手の戦術に対応する」は別のスキル
- 対応戦術は、相手を観察・分析する力と手持ちの引き出しの広さで決まる
- 対応は大きく変える前に、頻度・配置・起点・Optionの優先順位から検討する
- 相手が消しているもの、与えているもの、自チームが使えるものの重なりを見る
- MODULE 1で学んだ優位設計の切り替えが、対応戦術の根幹になる
