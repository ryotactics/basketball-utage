# MODULE 5 - Lesson 2｜vs PnRカバレッジ


## 前のレッスンからのつながり

対応戦術の第1弾として、PnRをディフェンスがどう守るかに対して、オフェンスはどう崩すかを扱います。

MODULE 4 Lesson 3では、PnRカバレッジを「守る側」の視点から整理しました。Drop、ICE、Show（Hedge）、Blitz、Switch——それぞれがどんな意図で設計されていて、どこに強みと弱みがあるかを見てきました。

このレッスンは、その裏返しです。

ディフェンス側の意図を理解しているからこそ、オフェンス側が「どこを突けばいいか」が見えてきます。カバレッジの仕組みを知らずにカウンターだけ覚えても、現場で応用できません。ディフェンスが何をしようとしているのかを読む力が、このレッスンの根幹にあります。

同じカバレッジでも、オフェンス側の対応次第でアドバンテージの取り方が変わります。カバレッジに対して「正解の動き」は複数あり、状況と選手の特性によって使い分ける必要があります。


## このレッスンの目的

- 主要PnRカバレッジ（Drop / ICE / Show / Blitz / Switch）それぞれに対するオフェンス側の読みと対抗策を体系的に整理する
- ハンドラー・スクリーナー・スペーサーがカバレッジを読んで動きを変える仕組みを理解する
- カウンターを「Handler / Screener / Action」の3つのカテゴリで整理し、設計・指導に使えるようにする
- PnRを「使う」から「使いこなす」レベルへの橋渡しをする


## カバレッジカウンターの3分類

PnRカウンターを設計するとき、アプローチは大きく3種類に分けられます。この分類を先に理解しておくと、現場での引き出し整理が楽になります。

### 1. 直接的なカウンター

選手の判断でカバレッジをその場で崩す方法です。ポゼッションごとのアジャストがしやすく、相手のカバレッジに即座に反応できます。HandlerがDropを見てPull Upを打つ、ScreenerがRollを加速させてDrop Dの後ろを取るといった動きがこれにあたります。

強み： 相手が変えてきたときにも柔軟に対応できる
弱み： 選手個人の読みと判断力に依存するため、精度の個人差が出やすい

### 2. Actionによるカウンター

アクション自体がカバレッジをさせない、または困難にする設計です。選手がプレー中にリードを読む負荷が下がります。例えばRamアクションはScreener DをPnRに遅れさせてDropを難しくする。AI WedgeはICEをさせない構造を作る。アクションの設計段階でカバレッジ対策が組み込まれています。

強み： 選手の判断負荷が下がる。再現性が高い
弱み： 相手がアクション自体を対策してくると崩されやすい

### 3. 構造によるカウンター

アクションの構造そのものがカバレッジを無効化する方法です。Spain PnRはScreener DをBack Screenで縛るためDropの目的自体を阻害します。'short' PNR（1inからの短距離PnR）はTagに出るスペースを与えないためShowに強い構造を持ちます。選手の読みやコールに関係なく、構造的に優位が生まれます。

強み： 設計段階での優位性が高い。相手が認識する前にアドバンテージを作れる
弱み： 相手がスカウトしてくると対策されやすい。引き出しとして固定化しすぎると読まれる


## ディフェンス側の意図を読む：Handler Dの守り方

カウンターを理解するには、まずディフェンスがどう守ろうとしているかを知る必要があります。Handler D（ボールハンドラーを守るディフェンス）がスクリーンに対してどう動くかによって、オフェンス側の正解が変わります。

### Fight Over

Handler DがスクリーンをほぼOver方向で完全に抜けてHandlerについていく方法です。遅れがほとんど生じないため、オフェンス側にとっては最も守られる形です。ただし、スクリーンの質が高いと現実的に機能しにくく、すべての状況でこれを維持することは難しいため、軸にはしにくいカバレッジです。

### Over

Handler DがスクリーンをOver方向でかわします。Handlerに遅れてついていくため、その遅れのタイミングでオフェンス側はカウンターを仕掛けやすくなります。HandlerのShoot力が高くPull Upがある場合に採用されることが多い。

→ オフェンス側の読み： Overされたタイミングの遅れを使う。JailやSnakeへの布石

### Under

Handler DがスクリーンをUnder（下側）を通ってかわします。中への侵入をさせにくいのが目的です。Handlerのシュート力が低くRollが怖い場合に採用されます。

→ オフェンス側の読み： Pull UpやMid RangeなどHandlerの外角脅威で裏返す

### Push & Under

UnderしているときにScreener DがScreenerをPushして押さえつける方法です。RollのScreener脅威が高い場合に採用されます。Screenerの動きを物理的に制限しようとする守り方です。

→ オフェンス側の読み： Early RollやFlipで逃げる。押さえつけられる前に動く

### Weak Hand Direction

Handler Dが逆手方向にDirectionをかけてHandlerを誘導します。ただし必ずScreener DはDropして下がっておく（Reject Driveを守るため）のがセットです。

→ オフェンス側の読み： 誘導方向を拒否してReject Drive。Directionをかけているということは逆方向が空いている


### vs Drop

DropはScreener DがリムとScreenerの間に位置してリムを守る守り方です。HandlerのMid RangeやPull Upを許容する代わりに、ペイントを守ることを優先します。

### Dropへの基本戦略3つ

Dropを攻略するには、まず大きな方向性として3つの戦略があります。個別のカウンターはこの3つのいずれかに位置づけられます。

### 1. Force the Over

Handler DをOverさせて後ろに背負う状況を作り、2on1の局面を生み出す戦略です。HandlerはRejectなどを挟んで、Handler DをタイトにFight OverされないようにScreenを有効活用します。Handler DがOverしてくる=遅れてついてくることを前提に、その遅れを使って2on1を作るのが狙いです。

### 2. Big must Separate

BigはRollやPopで素早く離れることで、x5（Screener D）に一人で二人を守る負担を強制します。HandlerとScreenerのパスラインを確保し続けることが重要で、Screenerがリムそばに居続けるとDrop DのポジショニングでHandlerもScreenerも同時に視野に入ります。Screenerが離れることで守備側の範囲を広げる発想です。

### 3. Create an Angle

HandlerがただBigに真正面から突っ込むのでなく、Driveの角度を作ります。身長差で不利にならないようDriveの進路を変えたり、Rollへのパスでアタック方向をずらすことでDrop Dに一直線に対応されない状況を作ります。


### Handler Counter vs Drop

#### Mid Range Shot（基本装備）

Drop CoverageはHandlerのPull-Up / Floater / Runnerに対して構造的に弱い守り方です。これがないとDrop Attackingが成立しにくくなります。確率的要素が強くその日の調子に左右されるため、軸に置くのは危険ですが、これがないとDrop Dはリムそばで安心して構えられるため、基本装備として必須です。

指導現場では「Handlerにミドルが打てるか」がDrop攻略の前提になります。打てないHandlerがいる場合、Drop Attackingの手が著しく限られることをコーチは認識しておく必要があります。

#### Attacking the Big

高めのDropをしているx5（Screener D）に対してHandlerが直接アタックする方法です。Screener DのBigが鈍重でHandlerがスラッシャータイプの場合に有効です。ただし低いDropに無理やりやるとリムプロテクトの餌食になるため、Drop Dのポジションを見てから判断する必要があります。

#### Hostage / Jail

Screenに対してOver方向でHandler Dをかわして遅れてきたHandler Dを背中側に背負って一度止まる動きです。Screener Dに対する2on1の状況を強調します。Deep Dropに対して特に有効です。

Jailからの派生：
- Jail Pull Up / Floater / Runner — Drop Dを引き付けてから打つ
- Jail Step Back — 背負いながらStep Backでスペースを作る
- Jail Clearout — Screenerを使ってHandlerのドライブスペースを確保（後述：Drop攻略の標準装備）

指導のポイント： Jailは「止まって待つ」動きのため、選手がスピードを落とすことを躊躇するケースがあります。「止まることが次の動きへのセットアップ」という考え方を徹底する必要があります。

#### Snake

HandlerがScreenを使った後にHandler Dが引っかかっている状況で逆側へ切り返す動きです。Step UpのシチュエーションではHandlerがHandler Dなしでフリーな状態でNailへ向かえます。

Snakeからの派生：
- Snake Pull Up — 切り返した後のPull Up
- Snake Escape 3pt — Nail付近でステップアウトして3pt

指導のポイント： SnakeはHandler DがScreenに引っかかることが前提です。Handler DがDropに切り替えたときは機能しないため、Handler DのOver意識を確認してから使う判断が必要です。

#### Pocket Pass

Deep Drop DがHandlerに意識が向いたところでRollへパスを通す方法です。Bounce Passが主な手段になります。HandlerのMid Range Shotがあることで、Drop DをHandlerに引き付けてから逆を使うのがポイントです。Mid RangeとPocket Passはセットで機能します。

#### Reject Drive

Drop Dがまだ準備できていない状況（Empty Side PnRなど）でHandlerがReject方向へドライブします。対応力依存のカウンターのため安定的な軸としては機能しにくいですが、相手の準備が間に合っていない局面では積極的に狙えます。

#### Retreat Attack（Pause & Retreat）

ScreenerのPopに対してHandlerが一度止まり、Drop DがPopに対してRecoverしようとする動きの隙を狙ってリムへアタックします。「止まる → 追いかけてきた瞬間に加速」のリズムが鍵です。


### Screener Counter vs Drop

#### Roll Race

ScreenerがRollを使ってリムへ素早く走り、Drop Dの後ろ側を取る動きです。Screenerの脅威度が高いほどDrop Dは大きく下がらざるを得なくなり、結果としてHandlerがDrop Dに対して自由度高くプレイできる状況が生まれます。

指導のポイント： Roll Raceは「速さ」と「角度」の両方が重要です。リムへ向かう角度が悪いとDrop Dに正面から対応されます。Drop Dの後ろ側を取る動線を設計してください。

#### Late Lob

HandlerがDrop Dを引き付けている間にRollへLobを出す方法です。Mid Rangeの脅威があればHandlerを警戒してDrop DのRoll対応が遅れるため、そのギャップにLobを通せます。HandlerのMid Range ShootとLate Lobはセットで機能する関係です。

#### Early Roll

Handler DがScreenを予見してOver Stanceでかわそうとした瞬間、ScreenerがScreenをかけずに早めにRoll（Hit しない）することで2on1の状況を作ります。Screener Dがスクリーンに対応する前にすでにRollしているため、守備側の対応が一テンポ遅れます。

指導のポイント： Early RollはScreenerの読みが必要です。「Handler DがOverに来そうだ」と感じた瞬間にスクリーンをかけない判断ができるかどうかが精度を決めます。

#### Gortat / Clearout（Jail Clearoutは標準装備）

ScreenerがRoll Raceした後にDrop DをSealしてHandlerのドライブスペースを確保する動きです。SnakeとJailの局面で特に有効です。

特にJail ClearoutはDrop攻略として最も再現性が高い標準装備カウンターとして位置づけられます。Handlerが背後にHandler Dを背負っている（Jail）状況でScreenerがDrop DをSealすることで、Handlerのドライブスペースを物理的に作り出します。

指導のポイント： Gortat/ClearoutはScreenerが「Rollした後に止まってSealに行く」二段階の動きです。Rollだけして終わりにならないよう、SealまでをセットとしてScreenerに落とし込む必要があります。

#### Pick & Pop

ScreenerがPopに出ます。Drop Dがリムを守るために下がっているため、Popしてくるビッグへの対応が難しくセパレーションができます。BigのC&S（キャッチ＆シュート）力がある場合に有効な手段です。


### Action Counter vs Drop

#### Spain PnR

Screener DにBack Screen（Stack Screen）をかけてDropの目的を阻害します。Drop DがBackscreenを受けてしまうとリムを守るためのポジションが取れなくなるため、Drop自体が成立しにくくなります。構造的にDropを破壊するカウンターです。

#### Seal Traffic（short PNR + Clearout）

1in状態（ペイント内に一人いる状況）でshort PNRを設定し、その後1inの選手がClearout（short Clearout）します。Drop Dも巻き込む形でHandlerのドライブコースを確保できます。

#### Veer Attack

BSからPindownに入るプレーです。Drop DがHandlerとScreenerを一人で守る瞬間の弱点を突く設計になっています。


### vs ICE

ICEは、PnRのScreenを使ってHandlerをサイドライン方向に追い込むカバレッジです。Handlerをペイントへ侵入させず、スクリーンの助けを借りた中への突破を防ぐことを目的とします。

### Handler Counter vs ICE

- Reject Drive — ICEのDirectionを拒否して逆方向へドライブ。ICE自体の前提を壊す最も直接的な返し
- Dribble Down & Get — ボールをサイドに追い込まれた後、改めてハンドラーがDribble Downして別のアクションに接続する
- Get Reject Give & Go — Getアクションに対してRejectを仕込み、Give & Goで展開する
- Make Space & Flat Screen — スペースを作ってFlatスクリーンでICEを無効化する角度を作る
- Still Get Middle — ICEの圧力をかいくぐって中を取りにいく読み
- Snake after using Flip — FlipアクションでICEのDirectionが切り替わる瞬間にSnakeを仕掛ける
- Split — スクリーンの間を抜けてICEの守り方に関係なく中へ侵入する

### Screener Counter vs ICE

- Flat Flip（→ Snake） — ScreenerがFlatなスクリーン角度でFlipし、続けてSnakeへ接続する
- Butt Screen — 背中を使ったスクリーン。ICEでDirection縛りをされているHandlerに対してルートを作る
- Reject Drive & Short Roll — HandlerのReject Driveに合わせてScreenerがShort Rollで合わせる2on1構造
- PNP（Pick & Pop） — ICEでペイントに追い込まれる前にPopへ逃げてスペースを作る
- Early Roll — ICEのセットアップが完了する前にEarly Rollで2on1を作る

### Action Counter vs ICE

- Wedge BS — BackscreenによってICEのDirectionポジションを崩す
- AI Wedge — AI構造にWedge Screenを組み合わせてICEを構造的に排除する
- Empty 77 Reject — 片サイドをEmptyにした77アクションでICEに対するRejectを設計する
- Turn 77 — 77アクションにTurnをかけてICEへの対応を変える

ICE対応のポイント： ICEの根幹はHandlerをサイドに誘導することです。最も効果的な崩し方はその誘導を拒否するReject Driveですが、Handler DとScreener Dの連携が整っているとRejectにもDrop Dが待っています。Screenerのカウンター（Flat Flip / Early Roll）を組み合わせて守備の連携を分断することが重要です。


## vs Show / Hedge

ShowまたはHedgeは、Screener DがPnRの後に大きく飛び出してHandlerに圧力をかけるカバレッジです。HandlerにスペースとタイミングをHandlerからHandlerを遅らせ、Screener Dが戻るまでの時間を守備側が稼ぎます。

### Handler Counter vs Show

- Reject Drive — ShowのDを逆方向に捨てる。Show Dはスクリーン方向に飛び出す設計のためReject側はガラ空き
- Drive the Flip — ScreenerのFlipアクションに対してHandlerがドライブ。Hedge DがFlipの方向へ移動するため逆側が空く
- Attacking the Hedge（Outside Drive） — 飛び出してきたShow D（Hedge D）の外側を回ってドライブ
- Split — スクリーンの間を抜けてShow Dを無効化
- Stop & Shot the Recover — Show Dが戻ろうとするタイミングでフェイクをかけて止め、すき間でシュート
- Retreat — 一度引いてShow DのRecoverを引き出してからリムへ
- Shot Fake PnP / Short Roll Get — ショットフェイクでShow Dを飛ばせてから次の選択肢へ

### Screener Counter vs Show

- Short Roll — Show Dが飛び出した後の空いたペイントへScreenerがShort Roll
- Slip — スクリーンをかける前にリムへ走り込む。Show Dが準備できていない瞬間を突く
- PnP（ACD with Clearout via Pop） — PopしてACDアクションに接続。ClearoutでHandlerスペースを作る
- Flip — ScreenerがFlipすることでTagの責任が移行し、Screener DのShowが弱まる効果がある
- Screen Again — ShowされてもScreenerが再度スクリーンをかけてShow Dを崩す
- Gortat / Clearout — Show DをSealしてHandlerのドライブスペースを確保

Flipのメリット： FlipはTagの責任をDから別のDへ移行させます。加えてScreener DのShowが弱まる副次的な効果があります。Show対策としてFlipは単なるオプションではなく、守備の連携を分断する有効な構造です。

### Action Counter vs Show

- Horns Twist Step Up — HornsセットからTwist + Step Upでshow Dの動線を崩す
- Flare to Pick — Flareスクリーンを経てPickへ。Show Dが予測できない動線を作る
- Rip to Pick — Ripアクション → Pickへ接続。エントリーを変えてShowをセットアップさせない
- Chase — ChaseアクションによってShowを遅らせる
- Step Up — HighなPnRでShow DのポジションをStep Upで崩す
- Ram — Screener DをRamスクリーンで遅らせ、Showの準備を遅らせる
- DHO（Dribble Hand Off） — DHOでHandlerを変えてShow Dをリセットさせる
- Spain PnR — Screener DにBack ScreenをかけてShowを構造的に阻害
- 'short' PNR — 短距離PnRでTagへの対応スペースをなくす。Show/Hedge対策としても有効


## vs Blitz / Trap

Blitzは、Screener DがPnRに対してハンドラーへ向かい、Handler DとScreener Dが合わせてHandlerをトラップする守り方です。Showよりも圧力が強く、ボールを失うリスクが高くなります。

### Handler Counter vs Blitz

- Reject Drive — Blitzは前提としてScreener DがHandlerへ向かう設計のため、Reject方向はほぼ無人。最優先のカウンター
- Outside Drive — Blitzしてきた2人の外側を回る
- Split — 2人の間を抜ける。スペースが少ないためリスクはあるが成功すれば直接ペイントへ
- PNP Get — Popして距離を取ってから改めてGetan

### Screener Counter vs Blitz

- Short Roll — Blitzで空いたパスレーンへScreenerがShort Roll。ハーフウェイポジションでボールを受けてアタック
- Flip — ScreenerがFlipしてBlitzの守備連携を崩す
- Drive the Flip — FlipしたScreenerに対してHandlerがドライブしてBlitzの2人を引き付ける

### Action Counter vs Blitz

- Delay Get — タイミングをずらしてBlitzの準備を間に合わせない
- PNP Get — PnPとGetを組み合わせてBlitzへの対応を強制する

Blitz対応の優先順位： Blitzは圧力が強いため、まずReject Driveでシンプルに逃げることを最優先にします。Split / Outside Driveはリスクも伴うため、Reject Driveが完全に封じられた状況での次の選択肢として扱います。


### vs Switch

スイッチはスクリーンに対してDFがポジションを入れ替えて対応するカバレッジです。スイッチの種類によってカウンターが異なります。

### vs Under & Switch（BSUOM）

スイッチを含んだBSUOM（Back Screen Under + Over Mismatch）によって、スイッチ後のミスマッチを構造的に設計します。

### vs Switch Up（アップスイッチへのカウンター）

小さいDFが大きいオフェンス選手を守るスイッチアップの状況への対応です。

- Back Seal — 背後でSealing。小さいDFがガードできない位置でポストアップ
- Drive it Back — ミスマッチを抱えたまま逆側へドライブして対応を引き付ける
- Alley Drive — Alley（通路）を使ったドライブでSwitchしたDを縦に抜く
- Foot Race — スピードミスマッチを活かして速く動くことでDFを振り切る
- Slip & Lift — SlipからLiftで合わせの動きを作る
- Hot Stove / Touch — クイックタッチでボールを動かしてスイッチ後の混乱を突く
- Drive the Slip（Drive the Slip Pop、Drive the Slip in） — Slipに対してHandlerがドライブして合わせる

### vs Triple Switch（トリプルスイッチへのカウンター）

複数人が連続してスイッチする場合の対応です。

- Speed Mismatch ± Screen the Mismatch — 速さのミスマッチを活かす、あるいはそのミスマッチを使ってさらにスクリーンをかける
- Duck-in — ミスマッチが生まれた瞬間にペイントへDuck-inしてポストアップ
- Corner Ghost / Backdoor — コーナーからGhostカット / バックドアでスイッチの混乱を突く
- Skip（Skip HO、Skip Pass、Skip ACD） — スイッチが追いつかない遠距離へのSkipパスで展開
- Ghost系 — Ghostカットの派生でスイッチ後のカバレッジを攪乱
- Spain — Screener DにBackscreenをかけてスイッチの構造そのものを崩す
- Ram系 — Ramアクションでスイッチを遅らせる

### Action Counter vs Switch

- Ram Invert Flip — RamにInvert + Flipを組み合わせてスイッチを誘発→利用
- Nail Ghost — Nail位置からのGhostカット
- Slip — スクリーンをかけない瞬間にリムへ走る。スイッチの前提を崩す
- Ghost — Ghostカットでスイッチ後の守備連携の穴を突く
- Spain — 構造的にスイッチを阻害


## Aggressive Coverage（Show / Blitz / Hedge）共通の優先順位

ShowやBlitz、HedgeなどのAggressive Coverageに対しては、カウンターの試みる順番が重要です。順番を間違えると、ボールプレッシャーが強すぎてカウンターに繋げられず、Stealからのターンオーバーで速攻を許す本末転倒になります。

### 優先順位1：Reject Drive

最初に狙うのは必ずReject Driveです。

Aggressive Coverageの守り方では、Screener DがHandlerへ向かうためReject方向のスペースが大きく空きます。Reject Driveはシンプルかつ最も確実に有効な手段です。Handler DやHedge Dによるボールプレッシャーがかかる前に素早く逆方向へ動くことが要点です。

### 優先順位2：Drive the Flip

次に狙うのはScreenerのFlipに対するDrive the Flipです。

ScreenerがFlipアクションを使うとHedge Dが逆側へ移動します。Handler DはDirectionで縛っている方向があるため、FlipによってHedge Dが動いた逆側へは守れない状況が生まれます。この瞬間を使ってHandlerがドライブします。

## 優先順位3：その他のカウンター

Reject DriveとDrive the Flipを試みてからその他のカウンター（Slip、Short Roll、Spain PnRなど）へ進みます。

なぜこの順番か：

Reject Driveを先に狙わないと、Handler DやHedgeによるボールプレッシャーが強すぎてその他のカウンターに繋げられません。プレッシャー下での複雑な動きはターンオーバーを招きます。まず最もシンプルで有効な手段から試みることで、守備側に「Rejectを守ること」と「他のカウンターを守ること」の両立を強いることができます。


### vs Tag

Tagとは、Screenerへのパスコース（特にRollへのパスコース）を守るためにHelp DFの選手がパスレーンに位置する動きです。

Tagへのカウンターは主にScreenerの位置変更で対応します。RollからPopへの切り替え（Pick & Pop）や、RollのラインをTagから外れたコースに変更することでパスコースを確保します。また、Tagに出た選手の元の守備ポジションが空くため、そこへのパスで展開する方法（Kick Out）もあります。

Tagが出づらい構造としては、'short' PNRがあります。ペイント内からのPnRはTagに出るスペース自体がなくなるため、構造的にTagを阻害できます。


## コーチが意識すべきこと

### ハンドラーとスクリーナーの共通言語を作る

カバレッジを読む力は、2人の間に共通言語があって初めて機能します。「Drop来てる」「Show来てる」「Snakeいける」などのコールが試合中に飛び交えば、読んだ情報が素早くアクションに変わります。このコールの体系をチームとして整備することが、PnR精度を上げる最初の一歩です。

### 5人全員がカバレッジに連動する

PnR対応は「2人の問題」ではなく「5人の読み合わせ」です。ハンドラーが正しく読んでも、残り3人のスペーサーの立ち位置が合っていなければ次のパスが出せません。

具体的には：
- Drop Dを引き付けてPocket Passを出すとき、Screenerがすでに良いRollラインにいるか
- Show DをReject Driveで抜くとき、コーナースペーサーがパスレーンを開けているか
- Jail Clearoutを使うとき、ScreenerがSealポジションに入るタイミングとHandlerのドライブタイミングが合っているか

### 確定的なカウンターを軸に置く

カバレッジ攻略において、確率依存のカウンター（Mid Range Shotなど）は「あればいい」ものですが、軸にはしてはなりません。Jail ClearoutやRoll Raceのように再現性の高い確定的なカウンターを軸に置き、確率的な要素はそこにプラスアルファで乗せる設計が現場での安定につながります。

### カウンターの3分類を意識して設計する

直接カウンター、Actionカウンター、構造カウンターのバランスを意識してください。直接カウンターだけでは選手の読みへの依存が高すぎます。Actionと構造のカウンターを組み合わせることで、選手の判断負荷を下げながら攻略の再現性を高めることができます。

### DropにはJail Clearoutを標準装備として教える

Dropに対する最も再現性の高いカウンターはJail Clearoutです。これは選択肢の一つではなく「標準装備」として位置づけ、PnRをやるすべての選手に身につけさせることを推奨します。Jailの止まる動き、Screenerのタイミング、Clearoutによるスペース確保——この3点が揃ってこそ機能します。

### カバレッジが変わることを前提に設計する

試合中、相手はカバレッジを変えてきます。DropからShowへ、SwitchからICEへ、といった変更に対して都度崩れていては得点効率が安定しません。各カバレッジに対応するカウンターをある程度揃えておき、試合中にアジャストできるハンドラーとコーチングを組み合わせることが現場での強みになります。


## 次のレッスンへ

次は、マンツーマンディフェンス全般に対してオフェンスはどう崩すか——vs 対人ディフェンスを扱います。PnRだけでなく、オフボールの動きやカッティング、ISOなど対人ディフェンス全般への対応に視野を広げます。


## まとめ

- 各PnRカバレッジには必ずオフェンス側の正解がある。ディフェンスの意図を理解することで正解が見えてくる
- カウンターは「Handler / Screener / Action」の3カテゴリに分類して設計する
- カウンターのアプローチは「直接的 / Actionによる / 構造による」の3種類があり、バランスが重要
- Drop攻略の標準装備はJail Clearout。再現性が最も高いカウンターとして軸に置く
- Aggressive CoverageへはReject Drive → Drive the Flip の順に試みる。この順番を守ることでボールプレッシャーに飲まれない
- ハンドラー・スクリーナーが読んで動きを変えられるとカバレッジを崩せる状態になる
- スペーサー含む5人全員がカバレッジに連動している。PnRは2人の問題ではなく5人の問題
