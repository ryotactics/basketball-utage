# MODULE 5 - Lesson 3｜vs 対人ディフェンス


## 前のレッスンからのつながり

前のレッスンでは、PNRカバレッジに対するオフェンス側の対抗策を扱いました。

このレッスンでは、より広い論点——マンツーマンディフェンス全般に対してどう攻めるかを整理します。

相手のマンツーマンの強度・スタイルに応じて、どのアクション系統や優位設計を選ぶかが問われます。


## このレッスンの目的

- マンツーマンの強度・スタイルに応じたオフェンスの選択肢を整理する
- ディフェンスの「強い部分」と「弱い部分」の読み取り方を理解する
- 自チームの引き出しを相手のマンツーマンに対して最適配置する発想を持つ


## 原理（MODULE 1との接続）

マンツーマンに対するオフェンスは、Opponent Weak の発見と設計が核になります。

- 守備の強度が高いなら → モーションで連続的に揺さぶる
- 特定選手のマッチアップが弱いなら → そこを狙い撃つ（PNRスイッチ誘発、ポストアップなど）
- Switchを多用するなら → スイッチ後のミスマッチを設計する

どこにOpponent Weakがあるかを見つけ、そこに火力を集中させるのが対人ディフェンスに対するオフェンスの基本思想です。


## 本文 — ユーザー執筆

このセクションでは、以下の観点から解説します。

### 1. 相手マンツーマンのスタイル別対応
- Aggressive Man（プレッシャーが強い）→ Back Cut / DHO
- Pack Line（ドロップ気味・ペイント防御） → 3P活用、スクリーンで外に連れ出す
- Switch All → ミスマッチISO、ポストアップ
- Deny強め → Backdoor、Flash、DHO、Screen Away
- Help Heavy → Drive & Kick、Skip、Extra、45 Cut
- No Help / Stay Home → 1on1、Post、PNRで個の優位を押し切る
- 相手の守り方を「強い／弱い」ではなく「何を消して何を与えているか」で読む

### 2. 守備者の個人特性を突く
- Hunting（狙った選手へボールを集める仕組み）
- 身長差のミスマッチ活用（ポストアップ）
- 速度差のミスマッチ活用（ドライブISO）
- ファウルトラブルの選手を守備に関わらせる
- Closeoutが弱い選手をHelpに出させてから攻める
- Screen Navigationが弱い選手を連続スクリーンに巻き込む
- Target HuntingとMatch Up Makingの違いをここで軽く予告する

### 3. ヘルプディフェンスを崩す
- ドライブ → Extra Pass でヘルプを動かす連鎖
- 逆サイドのスペーサーを活かすスキップパス
- Cut によるヘルプの吸引と逆サイドの空白利用
- Tagに出る選手を罰するSlip、Lift、Corner Cut
- Nail Helpを動かすためのSlot Drive、DHO、Post Flash
- Helpを出させないためのSpacingと、Helpを出させて罰するSpacingを分ける

### 4. 対応を選ぶ順番
- まず既存のオフェンス内で、頻度と配置を変えて対応できるかを見る
- 次に同じアクション内のOption / Readを変える
- それでも足りなければCounterや別セットを使う
- 最後にコンセプト自体を変える。小さい変更から始めることを強調する


## コーチが意識すべきこと

- 試合前に相手のマンツーマンのスタイルと弱点を特定できているか
- 試合中に「相手がどこを消してどこを与えているか」を読んでいるか
- その読みをハーフタイムまでに言語化してチームに共有できるか

対人ディフェンスへの対応は、スカウティングの深さ × 引き出しの広さで質が変わります。どちらかだけでは不十分です。
