## ディフェンスシステム構築のフロー

MODULE 3 Lesson 3で紹介したコンセプト設計のフローを、**システム設計の文脈でより具体化**します。

```
Step 1：ディフェンスコンセプトの策定
  └ 何を最優先で守るか（リム / オープン3P / アウトナンバー）
  └ 全体システム（マンツーマン / ゾーン / ハイブリッド）

Step 2：ボール方向の設定（On Ball Direction）
  └ No Middle（ペイント優先）or No Base（コーナー優先）
  └ コンタクトとインラインの基準

Step 3：ヘルプシステムの決定
  └ Help & Sink / No Help / Stunt & Recover / Gap Help / Tandem
  └ ボールラインの設定

Step 4：PnRカバレッジの策定
  └ 主カバレッジと控えカバレッジ
  └ タグ（Tag）の設計

Step 5：ローテーションの策定
  └ X-Out / Peel のどちらを基本とするか
  └ スクランブル（全崩れからの立て直し）の共通ルール

Step 6：特殊ディフェンスの策定
  └ Top Lock（特定シューターへの事前対応）
  └ Trap / Double Team の発動条件
  └ ゾーン / プレスの運用基準
```

## ディフェンス指標との連動

ディフェンスシステムを設計するだけでなく、**機能しているかを測る指標**も持つ必要があります。

代表的なディフェンス指標：

- **Defensive Rating（DF Rating）** — 100ポゼッションあたりの失点
- **Paint Points Allowed** — ペイント内での失点数（リム保護の指標）
- **Open 3P% Allowed** — コンテストのない3Pシュートの被成功率
- **Transition Points Allowed** — トランジションでの失点（帰陣・ヘルプの指標）
- **Deflection（ディフレクション）** — プレッシャーの強度を示す数値

指標を持つことで、「守れていない」という感覚論から、「何が問題か」という構造的な分析に変わります。

---

## 最小修正で考える

ディフェンスシステムが崩れたとき、いきなり全体を変える必要はありません。まずは最小修正で解決できるかを見ます。

個人戦術を変える。Matchupを変える。ラインナップを変える。PnR Coverageを変える。特定アクションへの守り方を変える。Help / Rotationを変える。最後に、システムそのものを変える。

この順番を持っておくと、試合中の修正が整理されます。たとえばPnRでやられているからといって、すぐZoneに変える必要はありません。Handler Dの通り方、Screener Dの高さ、Tagの担当、CornerへのCloseoutのどこが崩れているのかを先に見ます。

システム変更は効果が大きい一方で、選手の認知負荷も大きい。だからこそ、普段から「どのレイヤーを変えているのか」を言語化しておくことが大切です。

---

## オフェンス → ディフェンスの連動設計

ディフェンスシステムは、オフェンスシステムと連動して設計する必要があります。

特に重要なのが**Transition（トランジション）の一体設計**です。

- オフェンスのリバウンド体制（OREBを狙うか、即帰陣か）は、**ディフェンスのトランジション強度と直結**します
- OREBを狙う設計なら → トランジションDFが薄くなるリスクを受け入れる
- 即帰陣を優先するなら → OREBを諦める

この選択も**オフェンス戦略 × ディフェンス戦略のトレードオフ**として明示的に決めておくべきです。

---

## 特殊ディフェンスの運用基準

ゾーンやプレスは「逃げの守備」ではなく、**戦略的に使う武器**として設計します。

| 特殊DF | 使うタイミングの例 |
|---|---|
| ゾーン | 相手がモーション中心で外のシュートが弱い / 自チームの個人DFが苦しい選手がいる |
| プレス | 相手のボールハンドリングが弱い / 点差・時間を考慮してリズムを崩したい |
| Top Lock | 特定シューターへのDenyを徹底してボールを持たせない |
| Trap / Double | ボールマンをハントして強制的にターンオーバーを狙う |

これらの**使う基準と切り替えの合図**をチームで決めておくと、試合中の混乱が減ります。

## コーチが意識すべきこと

- ディフェンスシステムの各ステップが**言語化されて選手に伝わっているか**
- ディフェンス指標を**シーズンを通じて観察しているか**（何が問題かを数字で見れるか）
- オフェンスとディフェンスのTransition設計が**一体で考えられているか**
- 特殊DFは「追い詰められたとき」ではなく、**戦略的な武器として使えているか**

## まとめ

- ディフェンスシステムは**コンセプト → ボール方向 → ヘルプ → PnRカバレッジ → ローテーション → 特殊DF**の順で設計
- **指標（Defensive Rating / Deflectionなど）** を持って機能度を測る
- Transition設計はオフェンスとディフェンスを**一体で考える**
- 特殊DFは**戦略的に使う武器**として運用基準まで設計する
