# MODULE 7 - Lesson 2｜スタッツ分析の基本


## 前のレッスンからのつながり

前のレッスンで、試合分析の3レイヤー（スタッツ → 映像 → 改善サイクル）を整理しました。

このレッスンでは第1レイヤー——スタッツ分析を扱います。

スタッツは「何が起きたか」を数字で把握するための手段です。感覚論ではなく、数字から問題のエリアを絞り込みます。


## スタッツ分析の役割

スタッツ分析でできること：

- 全体傾向の把握（どのエリアで点が取れているか / 取られているか）
- 問題の特定（どのアクション / シチュエーションで効率が落ちているか）
- 改善の効果測定（前の試合と比べて変化があったか）

スタッツ分析でできないこと：

- 「なぜ」の説明（数字は結果だけを示す。原因は映像で確認する）
- 個別の判断の良し悪し（流れや状況を含まない数字では判断しにくい）

スタッツは「何を見るか」の道しるべであり、答えではありません。


## オフェンス系の主要スタッツ

### 効率指標

指標 / 定義 / 見方
eFG%（実効FG率） / 3Pを1.5倍重みづけした野投成功率 / シュート選択の質を測る。FG%より実態に近い
TS%（真の成功率） / FG・3P・FTをすべて含む総合的な得点効率 / スコアラーの総合効率の比較に使う
Offensive Rating / 100ポゼッションあたりの得点 / チームオフェンス全体の効率指標
PPP（Points Per Possession） / 1ポゼッションあたりの得点 / アクション / シチュエーション別の効率比較に便利

### シュートエリア別

- ペイント内得点 / 試投数 → ドライブとポストアップの効率
- ミドルレンジ（FT〜3Pライン） → 2P効率の中でも価値が低いエリア
- 3P試投数・成功率 → 外のシュートの量と質

### ターンオーバー・リバウンド

- TOV%（ターンオーバー率） → 100ポゼッションあたりのTOV
- OREB%（オフェンスリバウンド率） → 取れるリバウンドのうち何%を取ったか


## ディフェンス系の主要スタッツ

| 指標 | 定義 | 見方 |
| --- | --- | --- |
| Defensive Rating | 100ポゼッションあたりの失点 | チームディフェンス全体の効率指標 |
| ペイント内失点 | ペイントエリア内で取られた点 | リム保護の強度を測る |
| オープン3P被FG% | コンテストのない3Pシュートの被成功率 | クローズアウトとスペーサー管理の質 |
| DREB%（ディフェンスリバウンド率） | 相手のFG非成功後に確保したリバウンドの率 | セカンドチャンスを与えていないかの指標 |
| Deflection（ディフレクション） | プレッシャーによるボールへのタッチ数 | ディフェンスの積極性・強度の代理指標 |


## スカウティングに使うスタッツ

相手チームのスカウティングでは、以下の数字が特に有効です。

- 相手のシュートエリア傾向 → どこから多く打ってくるか（3P重視 / ペイント重視）
- 相手のPPP by アクション → PNR / ポストアップ / アイソレーションのどれが高いか
- 特定選手のeFG%・使用率 → どの選手が得点の何%を担っているか
- TOV誘発率 → プレッシャーに弱い選手はどこにいるか

これらを把握することで、Opponent Weak（どこを狙えるか）と守るべき優先事項（どこを消すか）が絞り込めます。


## スタッツの使い方の注意点

### サンプルサイズに注意

1試合・1クォーターのスタッツは偶発的なブレが大きいです。複数試合の傾向を見ることで精度が上がります。

### 「文脈」を無視しない

数字だけで判断しない——例えばターンオーバーが多くても、それがプレッシャーをかけて攻撃的に動いた結果であれば問題ではないこともあります。

### 映像と必ず組み合わせる

スタッツで「どのエリアに問題があるか」を絞り込んだら、必ず映像でなぜそうなったかを確認する。スタッツだけで修正方針を決めると、原因の見当違いが起きます。


## 数字は「現象」、映像は「理由」を見る

スタッツは、何が起きたかを示します。しかし、なぜ起きたかまでは教えてくれません。

たとえばTOV%が高いとします。原因は、相手のPressかもしれません。PNRのBlitzに対する出口がないのかもしれません。単純なハンドリングミスかもしれません。Spacingが悪く、パスコースがなかったのかもしれません。数字だけでは、どの修正が必要かは決まりません。

逆に、映像だけを見ると印象に引っ張られることがあります。目立つミスが強く記憶に残り、実際には少ない頻度の問題を過大評価することがあります。

実務では、スタッツで異常値や傾向を見つけ、映像で原因を分類し、練習やゲームプランに落とし、次の試合で同じ指標を追う、という流れが使いやすいです。

数字と映像は対立するものではなく、役割が違います。数字で入口を見つけ、映像で原因を掘る。この組み合わせが分析の基本です。


## コーチが意識すべきこと

- 自チームが定期的に取れているスタッツは何か（選手がいる環境では取れる数字に限界がある）
- スタッツを「問題の絞り込み」に使っているか、「成果の確認」だけに使っていないか
- 相手スカウティングで、スタッツから優先的に攻める場所を特定できているか

次のレッスンでは、スタッツで絞り込んだ問題を映像で確認する——映像分析の方法を扱います。


## まとめ

- スタッツは「何が起きたか」を示す道しるべ。「なぜか」は映像で確認する
- オフェンスはeFG% / PPP / エリア別、ディフェンスはDefensive Rating / ペイント失点 / Deflectionが主要指標
- スカウティングでは相手のシュートエリア傾向・使用率・得点源を特定する
- 複数試合の傾向を見る習慣を持つ
