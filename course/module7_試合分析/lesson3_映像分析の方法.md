# MODULE 7 - Lesson 3｜映像分析の方法

---

## 前のレッスンからのつながり

前のレッスンでは、スタッツ分析の主要指標と使い方を整理しました。

スタッツは「何が」を示します。「なぜ」を明らかにするのが、このレッスンで扱う**映像分析**です。

---

## 映像分析の役割

映像分析でできること：

- **現象の原因を確認する**（なぜその数字になったのか）
- **戦術的な問題 / 判断の問題 / 実行の問題を区別する**
- **相手の傾向・パターンを具体的に把握する**（スカウティング）
- **選手へのフィードバック素材を作る**（映像で見せることで言葉よりも伝わりやすい）

映像分析でできないこと：

- **全体傾向の定量的な把握**（それはスタッツが担う）
- **主観を排した客観的な判断**（見る人の視点でバイアスがかかる）

---

## 映像分析の視点

試合映像を見るとき、以下の視点で観察します。

### 自チーム分析

| 観点 | 問い |
|---|---|
| **アクションの実行** | 設計した動きが正しく実行されているか |
| **判断** | 選手はその状況で正しい判断（Shoot / Drive / Pass）をしているか |
| **ポジショニング** | スペーシングは保たれているか。ヘルプポジションは適切か |
| **コミュニケーション** | スクリーンのコールやローテーションの指示が出ているか |
| **ミスの原因** | ターンオーバー / 失点の原因は戦術か、実行か、判断か |

### 相手スカウティング

| 観点 | 問い |
|---|---|
| **頻度の高いアクション** | どのPnR / セット / アクションを最も多く使うか |
| **得点のパターン** | 誰が、どの形で、どのエリアで得点しているか |
| **弱点** | どのカバレッジに対してミスが出るか。プレッシャーに弱い選手はどこにいるか |
| **ディフェンスの傾向** | Switch / Drop / HedgeなどPnRカバレッジはどれを主に使うか |
| **ATO / SLOB** | タイムアウト明けやスローインでどんなセットを使うか |

---

## 映像を見る前に「問い」を立てる

映像を漠然と見ていると、目についたことだけを記録する「ただの観察」になります。

**映像を見る前に「今日は何を確認するか」という問いを立てる**ことで、分析が目的を持ったものになります。

例：
- 「今日は自チームのPnRカバレッジの実行精度を確認する」
- 「今日は相手の3Pシューターがどこから打っているかを確認する」
- 「今日は自チームのオフボールのスペーシングを確認する」

1回の分析セッションで確認する「問い」は**1〜3つに絞る**ことで、時間効率が上がります。

---

## 1回目と2回目で見るものを変える

映像分析は、1回で全部を見ようとすると浅くなります。目的に応じて、見る回を分ける方が実務的です。

1回目は、試合全体の流れを見ます。どの時間帯で流れが変わったのか、どのラインナップで良かったのか、相手がいつ守り方を変えたのか。ここでは細部に止まりすぎず、試合の構造を掴みます。

2回目は、問いを絞って見ます。PnR Coverageだけを見る、Transition Defenseだけを見る、OREB後の配置だけを見る、Zoneに対する攻め方だけを見る。焦点を絞ることで、同じ現象が何度起きているかを確認できます。

3回目以降は、必要に応じて選手別・ユニット別に見ます。個人の判断ミスなのか、チームルールの未整備なのか、ラインナップの相性なのかを分けます。

映像分析は、見た量よりも、問いと分類の精度が重要です。

---

## 学習フェーズを分けて見る

映像を見るとき、最初から「自分ならどう設計するか」まで考えようとすると、かえって観察が濁ることがあります。

特に学習段階では、まずは守破離の「守」として、目の前のチームが何をしているのかを正確に観察することが大切です。どの選手のどんなプレーを生かしているのか。そのためにどのアクションを使っているのか。TransitionからEarly、Setへどう繋いでいるのか。相手のCoverageに対して頻度をどう変えているのか。

この段階で「もっと良い設計を考えなければ」と焦ると、観察より評価が先に立ちます。まず大量にインプットし、戦略と具体的なアクションの結びつきを見る。その蓄積が増えてから、自分ならどう応用するかを考える方が、分析は深くなります。

映像分析は、創作の前に観察があります。観察の精度が上がるほど、自分の設計も具体的になります。

## クリッピングとメモの方法

映像分析を習慣化するためには、**素早くクリップ・メモできる仕組み**が必要です。

### クリッピングの基準

- **判断の良い / 悪いシーン**（学習素材として選手に見せる）
- **パターンが繰り返されているシーン**（相手の癖 / 自チームの癖）
- **アドバンテージが生まれた瞬間 / 失われた瞬間**

### メモのポイント

- 「**何が起きているか**（現象）」と「**なぜそうなったか（原因）**」を分けてメモする
- 時間を止めて「分析」と「メモ」を同時にすると効率が落ちる——まず流して観察し、気になったシーンをクリップ、後でメモするほうが速い

---

## 個人分析とチーム分析

映像分析は**個人レベル**と**チームレベル**で行います。

### 個人分析

- 特定選手のシュート選択・ドライブ判断・オフボールの動きを確認
- 「この選手はどういう状況でどんな選択をしているか」のパターンを把握

### チーム分析

- ヘルプローテーションの連動
- スペーシングとポジショニングの全体像
- アクションの「意図」が5人に揃っているか

個人の問題とチームの問題を区別することで、フィードバックの対象（個人 vs チーム全体）と練習設計が変わります。

---

## コーチが意識すべきこと

- 映像を見るとき、**問いを立ててから見ているか**
- 「良いシーン」と「悪いシーン」を**バランスよくクリップしているか**（悪いシーンだけでは選手の学習が偏る）
- 分析の結果を**選手に伝えるための素材（クリップ）** として使えているか
- 映像分析に**定期的な時間を確保できているか**（試合直後 / 練習前など）

## まとめ

- 映像分析は「**なぜ**」を明らかにする手段
- 映像を見る前に**「今日の問い」を1〜3つ立てる**
- 自チーム分析では**実行 / 判断 / ポジショニング**の3層で問題を特定する
- 相手スカウティングでは**得点パターン / 弱点 / ディフェンス傾向**を確認する
- 「良いシーン」と「悪いシーン」をバランスよくクリップして、選手への学習素材にする
